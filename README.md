# BSIS2033
# Text Adventure Game

## Project Overview
고등학교 재학 시절 2인 개발 팀으로 기획 및 제작한 텍스트 어드벤처 게임입니다. 유저의 선택에 따라 여러 가지의 경로로 분기되며, 최종적으로 다양한 엔딩에 도달합니다.

* **개발 환경:** Java / Kotlin
* **Repository 목적:** APK 디컴파일을 통해 스크립트 데이터 복원

---

## My Role: Narrative & Logic Designer
저는 프로그래밍 언어에 능숙했던 팀원이 코드를 구현할 수 있도록, **게임의 뼈대가 되는 논리 구조와 전체 스토리를 설계**하는 역할을 전담했습니다. 

**1. 상태 전이(State Transition) 로직 기획**
* 특정 씬(Scene)에서 유저가 선택지를 골랐을 때 다음 씬으로 어떻게 이동할지, 전체 노드(Node)의 흐름을 유향 그래프 형태로 기획했습니다.

**2. 조건부 분기 데이터 모델링**
* 단순한 A/B 선택을 넘어, 특정 아이템을 가지고 있거나 일정 시간이 지났을 때만 해금되는 '조건부 선택지'의 논리 구조를 설계하여 팀원에게 전달했습니다.

**3. 엣지 케이스(Dead-end) 및 무결성 검증**
* 분기가 많아짐에 따라 발생할 수 있는 논리적 오류(진행 불가, 무한 루프 등)를 방지하기 위해, 모든 루트를 직접 시뮬레이션하며 기획의 무결성을 테스트했습니다.
